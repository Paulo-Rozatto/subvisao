# Subvisor
Ferramenta simples para anotacao de dataset

## Instalação
1. Faça a instalação do Node.js ([blog da Alura explicando como instalar](https://www.alura.com.br/artigos/como-instalar-node-js-windows-linux-macos));
2. Faça o clone desse repositório;
3. Abra o terminal na pasta do projeto e execute: 
```
npm install
```

## Rodar
Sendo realizada a instação, rode no terminal: 
```
npm start
```
Por padrão, o projeto estará rodando em [http://localhost:1234](http://localhost:1234)

### Rodar - opção 2
1. Se já tiver feito o clone do repositório, faça o checkout para a branch gh-pages:
```
git checkout gh-pages
```
2. Se não tiver realizado o clone, pode-se baixar o código buildado como zip <a href="https://github.com/Paulo-Rozatto/subvisor/tree/gh-pages" target="_blank" >daqui</a>.
3. Independente do modo, abra o terminal na raiz do projeto e execute:
```
python -m http.server
```
Por padrão o projeto estará rodando em [http://0.0.0.0:8000/](http://0.0.0.0:8000/)
